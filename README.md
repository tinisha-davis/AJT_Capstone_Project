# AJT Capstone Project 
A prototype React/Vite webapp for NSC's merch store website. 

# Table of Contents
* [Getting started](#getting-started)
* [Prerequisites](#prerequisites)
* [Installation](#installation)
* [Running Locally](#running-locally)

## Getting Started
* Clone the repository

`git clone <AJT_Capstone_Project>`

* Navigate to the directory 

`cd <AJT-Capstone>`

* Install Dependancies

* Start the development server

`npm run dev`

* Open the app in your browser by folloing the outputted link

### Prerequisites
* Node.js v14.18 or 16+ is required

### Installation 
* In command line: 
    * Nav to the project's root

    `cd AJT-Capstone`

    * Run: 

    `npm install`

### Running Locally
* in command line: 
    * At the project's root run: 

    `npm run dev`
